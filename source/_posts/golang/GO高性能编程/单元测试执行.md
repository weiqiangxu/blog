### 一、项目

``` bash
├── go.mod
└── tool
    ├── net.go
    └── net_test.go
```

``` bash
module github.com/ovn-org/libovsdb

go 1.20
```

### 二、执行

``` bash
$ cd /home/libovsdb

# 执行包内部所有单元测试
$ go test github.com/ovn-org/libovsdb/tool

# 查看执行结果 -v
# 执行部分函数 -run <unit_test_func_name>
$ go test -run TestGetUser -v github.com/ovn-org/libovsdb/tool
```
